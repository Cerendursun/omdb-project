# 🎬 OMDb Movie Explorer

A movie search and discovery application built using the OMDb API.  
Users can search for movies and view detailed information such as title, year, genre, director, IMDb rating, and poster.

## ✨ Features

- Movie search system
- Movie detail view
- Trending and featured section
- Recent searches (LocalStorage)
- Genre / year / IMDb filtering
- Fully responsive design
